# MAISON — Luxury E-Commerce Demo

Demo de e-commerce de moda de luxo, desenvolvida como portfólio de UX/UI para apresentar habilidades em design de interfaces e experiência de compra.

---

## Páginas incluídas

- **Home** — Hero animado com destaque de coleção e produtos em destaque
- **Collection** — Grade de produtos com filtros e ordenação
- **Product Detail** — Página de produto com seleção de tamanho, cor e galeria
- **Cart** — Carrinho com gestão de quantidade, cupom de desconto e resumo do pedido
- **Checkout** — Fluxo completo com dados de entrega, pagamento e confirmação de pedido

---

## Funcionalidades

- Navegação entre páginas sem reload (SPA simulado)
- Sistema de carrinho funcional com adição, remoção e atualização de quantidade
- Cupom de desconto ativo: `SAVE10` (10% off)
- Frete grátis automático para pedidos acima de $150
- Seleção de tamanho e cor na página de produto
- Filtros e ordenação na página de coleção
- Menu mobile com overlay e animação de entrada
- Header fixo com backdrop blur
- Modal de confirmação para ações do carrinho e pedido
- Quick add direto da listagem de produtos
- Layout responsivo para desktop e mobile

---

## Tecnologias

- HTML5 semântico
- CSS3 com variáveis customizadas, Grid e Flexbox
- JavaScript vanilla — sem frameworks, sem dependências
- Google Fonts: Syne + DM Sans
